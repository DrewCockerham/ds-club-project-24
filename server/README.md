## Setup
Make sure you have Node.js downloaded.

# 1. Cd to where you want to put this project.
`cd ${directory}`

# 2. Git clone this repo.
`git clone https://github.com/DrewCockerham/ds-club-project-24`

# 3. Cd to the cloned repo.
`cd ${directory}`

# 4. Install Packages
`npm install`

# 5. Run the Code
`npm run scrape`
